Directory for tests including unit tests, integration tests, and acceptance tests.
# Testing

This directory contains tests for ensuring the quality and functionality of the "Shadow Walkers" game project.

## Unit Tests

- `unit/`: Directory for unit tests targeting individual components and functions of the game.

## Integration Tests

- `integration/`: Directory for integration tests verifying the interaction between different modules and systems.

## Acceptance Tests

- `acceptance/`: Directory for acceptance tests validating the overall behavior and functionality of the game.
