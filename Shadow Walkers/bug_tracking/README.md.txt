# Bug Tracking

This folder is used for managing bug reports and tracking issues related to the "Shadow Walkers" game project.

## How to Report a Bug

1. Navigate to the Issues tab in this repository.
2. Click on the "New Issue" button.
3. Provide a descriptive title and detailed description of the bug.
4. Include steps to reproduce the bug, expected behavior, and actual behavior.
5. Add relevant labels and assignees.
6. Click on the "Submit new issue" button to create the bug report.

## Bug Tracking Guidelines

- Search existing issues before creating a new one to avoid duplicates.
- Use clear and concise language when describing bugs.
- Provide screenshots or videos if applicable.
- Follow up on assigned issues and provide updates as needed.
