Directory for storing various art assets including concept art, character designs, and environment assets.
# Art Assets

This directory contains various art assets used in the "Shadow Walkers" game project.

## Concept Art

- `Concept_Art/`: Directory for storing concept art used as visual references during game development.

## Character Designs

- `Character_Designs/`: Directory for character designs including sketches, illustrations, and animations.

## Environment Assets

- `Environment_Assets/`: Directory for environment assets such as backgrounds, textures, and props.
